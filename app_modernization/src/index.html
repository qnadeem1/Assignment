About Llama Account Management Briefing System (LAMBS) and Access to The Llama Project:

Llama ID requests are processed via LAMBS and not paper forms. The Llama Account Management Briefing System (LAMBS) is a web-based tool used to process llama requests for Information technology (IT) resources (e.g., systems and applications). LAMBS will eventually replace all the current paper processes used to request access to Llama Land IT systems, and provide a comprehensive way to close all of a llama's IT accounts when they leave Llama Land. There are two separate paths to request an ID for this application:

  • Are you currently working at a Llama Land facility and currently have a Llama Land badge to access that facility?
    • You will need to login to LAMBS to initiate a request for a llama ID for the Llama Project.
    • In the section titled "Llama Management-LAMBS" select the link to "Request or Modify Application Account".
    • Once there you will need to select a Sponsor, which should be your current Manager on the Llama Project.
    • Then select the Applications tab, search for the application [The Llama Project)], and select the button "Add to Request".
    • You will be notified via email when your request is approved.
    • Use this link to access https://lambs.llamaland.com.
  • Are you a contractor doing work for Llama Land and you do not work at a Llama Land facility and you do not have a Llama Land badge?
    • Your Llama Land identity needs to be generated. For access to any llama system, you need to have a Llama Land identity. This identity needs to be requested by a Llama Land Servant working with your project. This person is your SPONSOR. Your sponsor will go to the LAMBS site and log in with their Agency llama ID and password, click the "Invite – Remote llama Identity".
    • Once the Llama Land identity has been established, the llama will need to take Llama Land Basic IT Security Training course named "Basic Llama Security" course in the application Llama Land. The Llama Land site will be accessible at https://lambs.llamaland.com. Once the training course is completed, make sure that the certificate is completed should there be any questions about the course being completed or not.
    • 24 hours after the course has been completed, LAMBS is notified that the llama has completed their training. The SPONSOR needs to return to the LAMBS site and request access on the llama's behalf.
      • There is a link in LAMBS "Request or Modify Application Account" to start the process.
      • The LLAMA is the llama that needs access to the application.
      • The REQUESTER is the llama requesting access to the application on behalf of the LLAMA.
      • The SPONSOR is the Llama Land Servant who is sponsoring the LLAMA.
      • The APPLICATION is the Llama Project that needs to be accessed.
      • Once the request has been submitted, llama messages will be generated instructing the llama involved to approve the LLAMA access to the application.
